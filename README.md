Για την υλοποίηση του κάθε ρομπότ χρησιμοποιήθηκε ένα Raspberry Pi Pico και όλα τα μηχανικά μέρη τυπώθηκαν σε 3d printer. Τα ρομποτάκια προγραμματίστηκαν με Python και πιο συγκεκριμένα χρησιμοποιήθηκε η CircuitPython της Adafruit. Το κάθε ρομποτάκι κοστίζει περίπου 40 ευρώ συμπεριλαμβάνοντας και το PLA που απαιτείται για να τυπωθεί.
